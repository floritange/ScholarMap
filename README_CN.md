## ⚡ 概述

**ScholarMap** 收集顶会的录用论文，提取作者与机构信息，构建 **力导向图谱** — 在浏览器中即可查看谁和谁合作、哪些机构最活跃、不同会议的社区如何关联。

## 📊 数据覆盖

| 会议 | 年份 | 论文数 |
|------|------|--------|
| **ICSE** | 2025, 2026 | ~400 |
| **FSE** | 2025, 2026 | ~350 |
| **ASE** | 2025 | ~300 |
| **ISSTA** | 2025 | ~130 |
| **合计** | | **1,266 篇论文、4,130 位作者、860 个机构** |

## ✨ 功能

1. **图谱视图**: 力导向布局，点击机构节点展开成员
2. **会议筛选**: 按会议切换，对比不同社区
3. **搜索**: 快速检索任意作者或机构
4. **统计面板**: 机构/作者排名，实时联动
5. **论文详情**: 标题、作者（标注一作/末作）、摘要、arXiv 链接
